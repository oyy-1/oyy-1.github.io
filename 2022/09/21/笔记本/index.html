  1. int main（）

  2. printf(“文本”)打印文本\n换行符

  3. Ctrl+F5运行 或 FN+Ctrl+F5（笔记本）

  4. int 是整形的意思
    main前面的int表示函数调用返回一个整型值
    return 0；返回0 （前后呼应）

  5. 数据类型
    char //字符数据型
    short //短整型
    int //整型
    long //长整型
    long long //更长的整型
    float //单精度浮点型
    double //双精度浮点型
    （小数称为浮点型）

  6. %c 打印字符格式的数据
    %d 打印整型十进制的数据
    %f 打印浮点型——打小数
    %p 打印地址形式

  7. 字节
    计算机中的单位
    bit——比特位
    byte-字节
    一个字节=八个比特位的大小

  8. 变量包括局部变量和全局变量
    main内称为局部变量
    main外称为全局变量
    {}称为代码块
    局部变量和全局变量尽量不要相同，当相同时，局部变量优先被选择

  9. scanf(“”,&XXX)输入函数 &取地址符

  10. 作用域
    变量哪里可以用哪里就是作用域
    extern 声明外部符号
